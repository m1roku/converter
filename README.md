## Project Description

Приложение, позволяющее посмотреть курс валют на сегодняшнюю дату, а так же сконвертировать одну валюту в другую.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
